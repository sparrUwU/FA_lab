Overfprintf:

Римские цифры:
  2023 = MMXXIII
  49 = XLIX
  99 = XCIX

Цекендорфово представление:
  20 = 01111
  15 = 010011
  7 = 0111

Системы счисления:
  255 в 16-ричной (нижний): ff
  255 в 16-ричной (верхний): FF
  42 в 2-ичной: 101010
  -100 в 8-ричной: 37777777634

Перевод из систем счисления:
  "ff" из 16-ричной: 255
  "FF" из 16-ричной: 255
  "1010" из 2-ичной: 10

Дампа памяти целых чисел:
  int 305419896: 01111000 01010110 00110100 00010010
  unsigned int 305419896: 01111000 01010110 00110100 00010010
  int -1: 11111111 11111111 11111111 11111111

Дампа памяти вещественных чисел:
  float 3.14: 11000011 11110101 01001000 01000000
  double 2.71828: 10010000 11110111 10101010 10010101 00001001 10111111 00000101 01000000

Комбинированный пример:
Число 42 = XLII (рим.) = 2a (16-рич.) = 01010001 (Цекенд.)
