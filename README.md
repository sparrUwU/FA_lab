# Фундаментальные алгоритмы и структуры данных на С

Коллекция учебных проектов и реализаций фундаментальных алгоритмов, структур данных и низкоуровневых компонентов на языке C.

## Содержание

Репозиторий включает:

* Алгоритмы работы с числами и системами счисления;
* Решето Эратосфена и задачи теории чисел;
* Реализации функций стандартной библиотеки C (`string.h`, `memory.h`);
* Пользовательские версии `printf`/`scanf` с расширенными спецификаторами;
* Динамический массив (`Vector`) на основе макросов;
* Бинарную кучу (`Heap`);
* Проверку корректности скобочных последовательностей;
* Интерпретатор математических выражений;
* Алгоритмы форматирования текста и обработки файлов;
* Преобразования чисел с использованием только битовых операций.

## Используемые технологии

* C17
* GCC / Clang
* Стандартная библиотека C
* Динамическое управление памятью
* Макросы и обобщённое программирование
* Побитовые операции
* Работа с файлами и строками

## Цель проекта

Репозиторий создан в рамках изучения языка C, алгоритмов и структур данных. Основной акцент сделан на самостоятельную реализацию базовых механизмов без использования готовых решений стандартной библиотеки.

## Структура

Каждая директория содержит отдельный набор задач с подробным описанием условий и реализованных решений.
