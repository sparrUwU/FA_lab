# Первый пак задач

### 1 задача 
Разработать консольную программу на языке C, которая получает через аргументы командной строки число x и флаг операции. В зависимости от переданного флага программа должна выполнять одну из следующих задач: выводить натуральные числа, кратные x, в диапазоне до 100; определять, является ли число простым или составным; представлять число в шестнадцатеричной системе счисления с выводом отдельных цифр; строить таблицу степеней для оснований от 1 до 10; вычислить сумму натуральных чисел от 1 до x; вычислять факториал числа x. Также необходимо обеспечить проверку корректности входных данных и обработку ошибок.
### Решение 
В программе реализовано преобразование строкового аргумента в тип long long с проверкой корректности ввода и переполнения. Для поиска кратных чисел используется последовательный перебор значений с шагом x. Проверка простоты числа выполняется перебором возможных делителей до квадратного корня из числа. Перевод в шестнадцатеричную систему реализован алгоритмом длинного деления строки на 16, что позволяет работать с числами произвольной длины. Таблица степеней строится последовательным умножением основания на себя. Сумма натуральных чисел вычисляется по формуле арифметической прогрессии с контролем переполнения, а факториал — последовательным перемножением чисел от 1 до x с ограничением по максимальному допустимому значению для типа long long. Программа возвращает соответствующие коды ошибок при некорректном вводе или переполнении.

### 2 задача 
Разработать программу на языке C, которая по порядковому номеру простого числа определяет его значение. На вход подаётся количество запросов T, после чего для каждого запроса вводится число n, обозначающее номер искомого простого числа. Для каждого запроса необходимо вывести соответствующее n-ое простое число.
### Решение 
Для решения задачи реализован алгоритм решета Эратосфена. При инициализации формируется массив всех простых чисел до заранее выбранной границы (2 000 000), после чего простые числа сохраняются в отдельный динамический массив. Для каждого запроса выполняется обращение к элементу массива по индексу n−1, что позволяет получать ответ за постоянное время. Дополнительно реализованы функции преобразования строк в целые числа с проверкой корректности ввода и возможного переполнения.

### 3 задача 
Разработать консольную программу, принимающую через аргументы командной строки флаг и набор параметров. В зависимости от флага программа должна выполнять одну из следующих операций: находить корни квадратного уравнения для всех уникальных перестановок коэффициентов (-q), определять кратность одного целого числа другому (-m), а также проверять возможность существования прямоугольного треугольника с заданными сторонами (-t). Необходимо контролировать количество и корректность переданных параметров.
### Решение
В программе реализована обработка трёх режимов работы. Для флага -q вычисляются корни квадратного уравнения для всех перестановок коэффициентов с использованием дискриминанта и сравнения вещественных чисел с заданной точностью ε. Для флага -m выполняется проверка кратности чисел; при работе с большими значениями используются признаки делимости для наиболее распространённых делителей, что позволяет избежать переполнения. Для флага -t стороны треугольника сортируются по возрастанию, проверяется выполнение неравенства треугольника, после чего по теореме Пифагора с учётом точности определяется, является ли треугольник прямоугольным. Во всех режимах предусмотрена проверка корректности входных данных и обработка ошибок преобразования чисел.

### 5 задача 
Разработать программу на языке C, которая получает через аргументы командной строки флаг и путь к входному файлу. В зависимости от флага программа должна выполнять обработку содержимого файла и записывать результат в выходной файл. Поддерживаются следующие режимы работы: удаление арабских цифр из текста (-d), подсчёт количества латинских букв в каждой строке (-i), подсчёт количества специальных символов в каждой строке (-s) и замена всех нецифровых символов их ASCII-кодами в шестнадцатеричной системе счисления (-a). Имя выходного файла может задаваться пользователем либо формироваться автоматически.
### Решение
Программа выполняет посимвольную обработку входного файла. Для режима -d из потока исключаются все символы цифр. В режиме -i для каждой строки подсчитывается количество символов латинского алфавита, а в режиме -s — количество символов, не являющихся латинскими буквами, цифрами или пробельными символами. Для режима -a каждый нецифровой символ заменяется его ASCII-кодом, записанным в шестнадцатеричной системе счисления. Обработка выполняется последовательно без загрузки файла целиком в память, что позволяет работать с файлами произвольного размера.

### 7 задача 
Разработать программу на языке C, которая считывает из входного файла числа, записанные в неизвестных системах счисления с основаниями от 2 до 36. Для каждого числа необходимо определить минимальное основание системы счисления, в котором его запись является корректной, перевести число в десятичную систему счисления и записать в выходной файл исходное число без ведущих нулей, найденное основание и его десятичное представление.
### Решение
Программа последовательно считывает числа из входного файла и приводит буквенные символы к верхнему регистру для унификации обработки. Затем удаляются ведущие нули и определяется минимально возможное основание системы счисления как значение, на единицу большее максимальной цифры, встречающейся в записи числа. После этого выполняется перевод числа в десятичную систему с помощью стандартной функции преобразования с указанным основанием. Для каждого корректного числа в выходной файл записываются исходная запись без ведущих нулей, минимальное основание системы счисления и соответствующее десятичное значение. В программе предусмотрены проверки корректности входных данных и обработка ошибок работы с файлами.
